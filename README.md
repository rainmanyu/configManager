# 借鉴自(pixle-vue-admin)

用于简单的集成管理


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```



### Deploy
```
1. compile for production
npm run build
2. node index.js
the deploy directory is dist
nohup node index.js &

URL:
ip:9888
```
